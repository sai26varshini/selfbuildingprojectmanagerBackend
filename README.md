#  Self-Building Project Manager (Backend)
```bash
This is the **backend** of the **Self-Building Project Manager Agent**, an AI-powered system that automates project planning, research, and management tasks using **FastAPI**, **MongoDB**, and **Groq API**.

---

##  Features

-  Built with **FastAPI**
-  Integrates with **Groq AI API** for AI-powered project generation
-  Uses **MongoDB Atlas** for database storage
-  Supports environment variables for secure API key management
-  Modular architecture with reusable agent scripts
-  Research paper generator and project workflow automation

---
```
